# Tallyit

It is a command line tool for taking notes about your financial balance.

## Installation

    $ gem install tallyit

## Usage

#### add a income tally

    $ tallyit --income -a 10000 -m "your msg about this tally" add

#### add a expend tally

    $ tallyit -a 10000 -m "buy something" add

#### show all records

    $ tallyit show

#### more info

    $ tallyit --help
